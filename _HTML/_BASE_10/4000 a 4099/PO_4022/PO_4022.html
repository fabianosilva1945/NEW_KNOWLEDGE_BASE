• CONTAGEM DA QUANTIDADE DE ITENS

Vamos pesquisar a quantidade de itens que contem na colection do banco de dados, usando o FIND().

Se você estiver usando uma versão mais recente do MongoDB (4.0+), o método .count() está obsoleto para consultas com filtros. O recomendado é usar: db.attendancesyncs.countDocuments()

  • db - è o banco de dados
  • ["attendancesyncs"] - è a colection
  • .find() - O método é usado para buscar documentos dentro de uma coleção no MongoDB. É como o SELECT * FROM tabela em SQL — ou seja, ele retorna todos os documentos da coleção, a menos que você especifique um filtro.
  • .count() - Metodo usado para contar o que tem em uma coleção.
	
➤ db["attendancesyncs"].find().count()
➤ db["attendancesyncs"].count()
