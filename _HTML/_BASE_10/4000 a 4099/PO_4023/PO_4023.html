• USANDO O DITINCT

O método distinct() é usado para retornar todos os valores únicos de um campo específico dentro de uma coleção. Esse comando retorna uma lista com todos os valores únicos encontrados no campo status da coleção. "Me mostra todos os valores diferentes que existem neste campo."

  • Quando usar distinct()
    • ➤ Saber quais valores diferentes existem em um campo.
    • ➤ Criar listas de opções (como status, categorias, tipos).
    • ➤ Fazer validações ou auditorias de dados.
				
					db["attendancesyncs"].distinct("code")

Observação: A descrição deve ser em minusculo ou gera erro na busca.
